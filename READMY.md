# Berkut (Беркут) — Тактична Інтерактивна Карта

«Беркут» — це веб-платформа на базі Python та Flask для візуалізації, моніторингу та аналізу тактичної обстановки в реальному часі. Проект розроблений для координації дій, нанесення військової номенклатури, побудови маршрутів та аналізу рельєфу (висот) з миттєвою синхронізацією змін між усіма підключеними клієнтами.

## 🚀 Основні можливості (Features)

* **Реалтайм Синхронізація (WebSockets):** Будь-які зміни на карті (створення, видалення, переміщення об'єктів або маршрутів) миттєво транслюються всім активним користувачам через **Flask-SocketIO**.
* **Тактичний Арсенал & Номенклатура:** Вбудований деталізований довідник об'єктів (жива сила, фортифікація, логістика, інженерні загородження, БПЛА, артилерія, бронетехніка, ППО, РЕБ та флот) з прив'язкою до умовних знаків.
* **Аналіз рельєфу (SRTM):** Локальне кешування та обробка висот за допомогою бібліотеки `srtm` — отримання висоти точки над рівнем моря та побудова профілю висот для маршрутів.
* **Керування маршрутами:** Створення, колірна диференціація та збереження тактичних маршрутів (треків) із послідовним індексуванням точок.
* **Безпека та Аудит:** * Рольова модель доступу (`admin`, `operator`).
    * Захист від брутфорсу (автоматичне блокування акаунту на 15 хвилин після 5 невдалих спроб входу).
    * Повний аудит дій системи (**Action Logs**) — логування кожної маніпуляції з об'єктами із прив'язкою до користувача, часу та координат.
* **Інтеграція з DeepStateMap:** Модуль для динамічного отримання актуальних геоданих лінії фронту через зовнішній API.

## 🛠 Технологічний стек (Tech Stack)

* **Backend:** Python 3.10+, Flask, Flask-SocketIO (WebSockets), Werkzeug Security
* **Database:** SQLite3 (швидка локальна БД з підтримкою `Row` фабрики для оптимізації вибірок)
* **Геодані:** Бібліотека `srtm` (локальний кеш матриці висот у `static/h`), Requests (інтеграція з зовнішніми API)
* **Frontend:** HTML5, CSS3, JavaScript (Leaflet.js / OpenLayers — *впиши свій варіант фронту*)
